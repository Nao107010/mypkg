# インストール手順
このリポジトリはROS2のパッケージなので、未インストール方はインストールをお願いします。
ROS2インストール後、以下のようにリポジトリをクローンしてください。
```
$ git clone https://github.com/Nao107010/mypkg.git
```

# 動作確認済み環境
以下の環境で動作確認がされています。
* OS Windows10
* Ubuntu 20.04
